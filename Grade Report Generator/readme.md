# Grade Report Generator

A simple JavaScript program that converts student scores into a grade report.

## Features

- Calculates letter grade from score
- Checks if the student passed or failed
- Provides feedback based on grade
- Generates a formatted student report in uppercase

## Functions

### `getLetterGrade(score)`

Returns the letter grade according to the score.

| Score | Grade |
|------|-------|
| 90-100 | A |
| 80-89 | B |
| 70-79 | C |
| 60-69 | D |
| Below 60 | F |

Example:

```js
getLetterGrade(92); // "A"
getLetterGrade(48); // "F"

```
